Приложение "Бомбила".
Бартерное такси.

В приложении используются следующие персональные данные пользователей.

* номер мобильного телефона,
* e-mail,
* имя и фамилия,
* изображения документов водителя для фотоконтроля,
* расположение пользователя по GPS,
* push-token устройства,
* данные BarterCoin-карты.

Все перечисленные данные передаются только на сервер приложения, и не передаются никаким третьим лицам.

Все перечисленные данные передаются по протоколу https.

Номер мобильного телефона используется в качестве идентификатора пользователя в системе. А также для дозвона средствами мобильного устройства.

Push-token устройства передаётся на сервер для хранения и передачи push-уведомления пользователю. Также push-token отправляется в Google Firebase, т.к. отправка push-уведомлений происходит через Google Firebase.

Данные BarterCoin-карты сохраняются локально в приложении. На сервер приложения передается только номер карты, без CVC кода для проведения платежа. Платёж проходит через внутренний сервер "Милитари Холдинга", на этот сервер непосредственно для платежа передаются все данные карты плательщика, в том числе CVC-код.

BarterCoin - это внутренняя валюта "Милитари Холдинга".

Соглашаясь с приведённым выше документом, пользователь передаёт компании "Милитари Холдинг" право использовать его персональные данные, указанные выше в документе, в указанном порядке.
